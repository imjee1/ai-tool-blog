오늘 Hacker News는 Anthropic의 새 모델 Claude Opus 5 발표가 압도적인 화제였다. 그 외에도 오픈 웨이트 모델로 비용을 크게 줄인 오디오북 생성 프로젝트, Hetzner의 LLM 추론 서비스 진출 소식 등 인프라·비용 효율화 쪽 관심도 눈에 띈다. AI 밖에서는 인도 정부의 Bitchat 앱 삭제 명령과 Patreon의 감원 소식이 각각 규제와 크리에이터 이코노미 이슈로 주목받았다.

## Claude Opus 5

Anthropic이 Claude 시리즈의 최신 플래그십 모델인 Opus 5를 공식 발표했다. 댓글 수만 659개에 달할 정도로 논쟁이 뜨거운데, 벤치마크 성능뿐 아니라 실제 코딩·에이전트 워크플로우에서의 체감 성능 개선 여부가 핵심 쟁점으로 다뤄지고 있다. 개발자라면 자신의 워크플로우에 실제로 얼마나 도움이 될지 직접 테스트해볼 가치가 있는 릴리스다.
👉 https://www.anthropic.com/news/claude-opus-5

## Show HN: Echo – 오픈 웨이트 모델로 Fable 수준 결과를 1/3 비용에

Echo는 오디오북/내레이션 생성 서비스인 Fable과 비슷한 품질을 오픈 웨이트 모델만으로 구현하면서 비용을 3분의 1 수준으로 낮춘 프로젝트다. 상용 API 대신 오픈소스 모델 조합과 파이프라인 최적화로 품질과 비용 사이의 균형점을 찾은 사례로, HN에서도 실제 오디오 샘플과 구현 디테일에 대한 논의가 활발했다. 폐쇄형 API 의존을 줄이고 싶은 개발자들에게 참고할 만한 실전 사례다.
👉 https://news.ycombinator.com/item?id=49026810

## Claude Cookbook

Anthropic이 공식적으로 공개한 Claude Cookbook은 프롬프트 엔지니어링부터 RAG, 에이전트 구축, 툴 사용 패턴까지 실전 예제 코드를 모아둔 레퍼런스다. 단순 API 문서를 넘어 실제 프로덕션에서 쓸 법한 패턴들을 정리해뒀다는 점에서 실용성이 높다는 평가가 많았다. Claude 기반 애플리케이션을 만드는 개발자라면 즐겨찾기 해둘 만한 자료다.
👉 https://platform.claude.com/cookbook/

## Hetzner, LLM 추론 서비스 준비 중

독일 클라우드 호스팅 업체 Hetzner가 자체 LLM 추론(inference) 서비스를 준비하고 있다는 소식이다. AWS·GCP 대비 저렴한 가격으로 유명한 Hetzner가 이 시장에 뛰어든다면 추론 비용 경쟁 구도에 변화를 줄 수 있다는 기대가 댓글에서 이어졌다. 자체 인프라에서 저비용으로 모델을 서빙하려는 팀들에게는 지켜볼 만한 움직임이다.
👉 https://sliplane.io/blog/hetzner-inference

## 인도 정부, Bluetooth 채팅 앱 Bitchat 삭제 명령

Jack Dorsey가 만든 인터넷 없이 Bluetooth 메시로 동작하는 채팅 앱 Bitchat에 대해 인도 정부가 보안 우려를 이유로 GitHub에 삭제를 명령했다. 탈중앙화·오프라인 통신 앱이 국가 차원의 규제 대상이 될 수 있다는 점에서 프라이버시 기술과 규제 사이의 긴장을 보여주는 사례다. AI 이슈는 아니지만 오픈소스 프로젝트가 정부 규제와 충돌하는 방식에 관심 있는 개발자라면 살펴볼 만하다.
👉 https://www.thehindu.com/news/national/government-orders-github-to-remove-bluetooth-based-chat-app-bitchat-over-security-concerns-jack-dorsey/article71262049.ece

## Patreon, 인력 20% 감원

크리에이터 후원 플랫폼 Patreon이 전체 인력의 20%를 감원한다고 발표했다. 창업자 Jack Conte가 직접 쓴 공지에서 시장 환경 변화와 비용 구조 재정비를 이유로 들었다. 크리에이터 이코노미 플랫폼들이 겪는 성장 정체와 구조조정의 흐름을 보여주는 사례로 주목받았다.
👉 https://www.patreon.com/jackconte/posts/painful-update-164628951

## HaikuOS에서 네이티브로 구동되는 Half-Life 2

Turing 아키텍처 GPU용 NVIDIA 드라이버를 HaikuOS에 포팅하는 프로젝트의 일환으로, Half-Life 2가 네이티브로 구동되는 데 성공했다는 소식이다. 오랜 기간 이어져온 오픈소스 OS 커뮤니티의 하드웨어 지원 확대 노력이 결실을 맺은 사례로, 레트로/틈새 OS 애호가들 사이에서 화제가 됐다.
👉 https://discuss.haiku-os.org/t/haiku-nvidia-porting-nvidia-driver-for-turing-gpus/16520?page=18

## "해킹해서" YC Startup School에 합격한 이야기
