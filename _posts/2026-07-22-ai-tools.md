### Kimi K3, Fable과 함께 SoTA 경쟁에 합류
Fireworks AI가 소개한 Kimi K3 모델이 최근 화제였던 Fable 모델과 견줄 만한 성능을 보여준다는 벤치마크 결과가 공개됐습니다. 두 모델 모두 특정 태스크에서 State-of-the-Art(SoTA) 수준을 기록했다고 하며, 오픈소스/상용 모델 경쟁이 계속 치열해지고 있음을 보여줍니다. LLM 선택지가 늘어날수록 개발자 입장에서는 비용 대비 성능을 따질 옵션이 많아진다는 점에서 주목할 만합니다.
🔗 https://fireworks.ai/blog/kimik3-fable

### Bento: HTML 파일 하나에 담긴 파워포인트
Bento는 편집, 뷰어, 데이터, 실시간 협업 기능까지 전부 단일 HTML 파일 안에 구현한 프레젠테이션 도구입니다. 별도 서버나 앱 설치 없이 브라우저만으로 슬라이드를 만들고 공유할 수 있다는 점이 특징입니다. "웹 표준만으로 얼마나 복잡한 앱을 만들 수 있는가"를 보여주는 좋은 사례로, 프레젠테이션 도구 시장에 던지는 신선한 접근입니다.
🔗 https://bento.page/slides/

### Anthropic, 저작권 침해 관련 15억 달러 합의금 판사 승인
Claude 학습에 사용된 불법 복제 도서와 관련해 Anthropic이 제기된 소송에서 15억 달러 규모의 합의금 지급이 법원의 승인을 받았습니다. 이는 AI 기업의 저작권 있는 콘텐츠 무단 학습에 대한 책임을 묻는 소송 중 역대급 규모의 합의로 기록됩니다. LLM 학습 데이터의 출처와 저작권 문제가 이제 실질적인 법적·재정적 리스크로 자리잡았음을 보여주는 사건입니다.
🔗 https://apnews.com/article/ai-anthropic-copyright-settlement-claude-books-bartz-74b140444023898aeba8579b6e9f0d63

### GigaToken: 약 1000배 빠른 토크나이저
GigaToken은 언어 모델의 토큰화(tokenization) 과정을 극단적으로 최적화해 기존 대비 약 1000배의 속도 향상을 주장하는 오픈소스 프로젝트입니다. 대규모 학습 파이프라인에서 토큰화는 종종 병목 지점이 되는데, 이 부분을 정면으로 공략한 접근입니다. 전처리 단계의 속도가 곧 학습 비용과 직결되는 만큼, 실제 벤치마크와 적용 사례를 지켜볼 가치가 있습니다.
🔗 https://github.com/marcelroed/gigatoken/

### 모나리자를 "그리는" GPT-5.6, Claude, Gemini, Grok 비교
색연필로 그린 것처럼 모나리자를 재현하도록 여러 최신 모델에게 시켜본 실험 포스트입니다. 각 모델이 이미지 생성/편집 지시를 얼마나 정교하게 이해하고 표현하는지 비교하며, 모델별 스타일과 한계가 뚜렷하게 드러납니다. 텍스트 기반 모델들의 시각적 이해·생성 능력 격차를 직관적으로 확인할 수 있는 재미있는 벤치마크입니다.
🔗 https://www.tryai.dev/blog/ai-drawing-arena-colored-pencils-claude-gpt-grok

### "가짜 면접 과제"였습니다: Git Hook에 숨겨진 악성코드
한 개발자가 받은 테이크홈 채용 과제를 직접 뜯어본 결과, 실제로는 정교하게 설계된 악성코드 배포 작전이었다는 사실을 밝혀낸 글입니다. Git hook을 이용해 클론 시점에 악성코드가 실행되도록 설계돼 있었으며, 겉보기엔 평범한 면접 과제로 위장돼 있었습니다. 구직 중인 개발자라면 받은 과제 코드를 실행 전 반드시 검토해야 한다는 경각심을 주는 사례입니다.
